# Extension Icons

This directory should contain the extension icons:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

## Creating Icons

You can create icons using any image editor. Suggested design:
- Red/white color scheme matching YouTube branding
- Simple remote control or play button icon
- Transparent background

For development, you can use placeholder icons from:
- https://via.placeholder.com/16x16
- https://via.placeholder.com/48x48
- https://via.placeholder.com/128x128

Or generate them using online tools like:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
